# Proces pisania pracy magisterskiej

## Opis projektu

Interaktywna prezentacja edukacyjna poświęcona procesowi pisania pracy magisterskiej. Projekt zawiera multimedialną prezentację z narracją audio, obrazami i interaktywnymi elementami, które prowadzą użytkownika przez wszystkie etapy tworzenia pracy dyplomowej.

## Zawartość

Prezentacja obejmuje kompleksowy przewodnik po procesie pisania pracy magisterskiej, od pomysłu do finalnej obrony.

## Struktura projektu

```
thesis/
├── index.html          # Główna prezentacja interaktywna
├── goodbye.html        # Strona końcowa
├── assets/            # Zasoby multimedialne (audio, obrazy)
│   ├── *.mp3         # Pliki audio z narracją
│   └── *.jpg/png     # Obrazy i grafiki
└── lib/              # Biblioteki JavaScript i CSS
    ├── player-*.js   # Odtwarzacz prezentacji
    ├── mondrian/     # Komponenty UI
    ├── rise/         # Moduły prezentacji
    └── sandbox/      # Style CSS
```

## Jak korzystać

### Wymagania

- Nowoczesna przeglądarka internetowa (Chrome, Firefox, Safari, Edge)
- JavaScript włączony

### Uruchamianie prezentacji

1. **Metoda 1: Otwieranie lokalnie**
   - Pobierz repozytorium
   - Otwórz plik `index.html` w przeglądarce internetowej
   - Prezentacja uruchomi się automatycznie

2. **Metoda 2: Hosting na serwerze**
   - Umieść pliki na serwerze WWW
   - Przejdź do URL-a gdzie hostowany jest `index.html`

### Nawigacja

- Używaj przycisków nawigacyjnych w prezentacji do poruszania się między slajdami
- Słuchaj narracji audio towarzyszącej każdemu slajdowi
- Prezentacja dostosowuje się do rozmiaru okna przeglądarki

## Technologie

Projekt wykorzystuje:
- **HTML5** - struktura prezentacji
- **JavaScript** - interaktywność i logika prezentacji
- **CSS3** - stylizacja i layout
- **Rise Player** - silnik prezentacji interaktywnych
- **LZ Compress** - kompresja danych
